## What you will do

In this project, you will visit the **Bug Hotel**: a web puzzle where each room contains
one tiny bug to fix.

Move from page to page by repairing small HTML, CSS, and JavaScript
mistakes. One room has a button that has slipped off-screen. Another room has a button
with the wrong function name. Later rooms hide clues with the wrong `id`, poor colour
contrast, or a broken link.
